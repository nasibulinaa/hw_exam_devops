# 1. Запись архитектурных решений проекта

Дата: 2025-11-29

## Статус

Принято

## Контекст

Нужно записать принятые архитектурные решения проекта.

## Решение

Используем условия из [задачи](https://lms.mipt.ru/mod/page/view.php?id=230712).

Кратко:

* Каждое решение принимается на основе анализа альтернатив
* Учитываются компромиссы между между простотой/сложностью реализации
* Решения документируются через соответствующий инструмент


## Последствия

Решения документируются через [adr-tools](https://github.com/npryce/adr-tools).
